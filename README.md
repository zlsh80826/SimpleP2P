# C++ / Simple peer to peer(P2P)
Use Google Prtocaol Buffer to implement C++ serialize,
and simple P2P function

## Client
* chat with peer
* download peer file
* kill yourself
* rigist
* login
* check online peer
* search online info
* beautiful color

#### Usage

```bash
./client <ip> <port>
```

example:

```bash
./client 192.168.1.3 8000
```

## Server
* check clint login info
* check online file
* beautiful color

#### Usage

```bash
./server <port>
```

example:

```bash
./server 8000
```
